# AI Session Transcripts

This directory contains Claude Code session logs from the development of ReviewLens AI.

Sessions are numbered chronologically. These are unedited working sessions including dead ends and iterations.

## Sessions

| File | Size | Description |
|---|---|---|
| `session-01-initial-setup.jsonl` | 19 KB | Initial project setup and exploration |
| `session-02-full-development.jsonl` | 28 MB | Full development session — P1 MVP, P2 features (evidence drawer, skill selector, insight report, image ingestion), promptfoo LLM eval, landing page, URL scraping, repo reorganization |

## Format

Files are in JSONL (JSON Lines) format — the raw Claude Code session log format. Each line is a JSON object representing a conversation turn including user messages, assistant responses, and tool calls.
